🚀 Automated Attendance Management System
📌 Overview
This project automates attendance tracking using Google Apps Script and Google Sheets.

The system controls sheet edit access using time-based triggers and stores attendance records in a centralized sheet.

⚙️ Features
✅ 8:30 AM – Edit access enabled automatically
✅ 9:15 AM – Edit access revoked automatically
✅ Centralized attendance storage
✅ Automated email confirmations
✅ Time-driven triggers (IST based)
🛠 Tech Stack
Google Apps Script (JavaScript-based)
Google Sheets
